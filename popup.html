How to Block Websites

Use simple patterns to block websites.

  • example.com → blocks exactly example.com
  • *.example.com → blocks all subdomains like www.example.com, blog.example.com
  • *example.com → blocks anything ending with example.com
  • example.* → blocks all top-level domains like example.com, example.org
  • example.com* → blocks anything on example.com like example.com/something and also matches example.com

Special Rules:

  • Use * to mean "anything"
  • No need to type http:// or https://
  • Matching is case-insensitive
